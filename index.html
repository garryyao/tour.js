Tour.js

Create a tour by moving a highlight box
between website elements

  • Launch example
  • or
  • Download v1.0

Setup

HTML

<p data-tour-desc="Element description"></p>

JavaScript

new Tour.Build('data-tour-desc');
new Tour.Build('data-tour-desc', {
  overlay: {
    opacity: 0.5
  },
  keyAccess: {
    previous: 'up',
    next: 'down'
  }
});
